Github+Hexo博客搭建

使用Hexo Github Netlify搭建博客

前置准备

  • 安装Node.js

  • 安装Git

创建Github仓库

  1. [创建仓库](Sign in to GitHub · GitHub) 用户名.github.io

  2. 生成并[配置SSH密钥](Sign in to GitHub · GitHub)

    ssh-keygen -t rsa -C "你的GitHub注册邮箱"

局部安装Hexo

  1. 安装到特定目录

    npm install hexo

  2. 添加环境变量（非必要）

    将目录\node_modules\.bin添加至系统环境变量下。如不添加环境变量则使用npx hexo init等命令。

  3. 初始化Hexo

    hexo init

配置Hexo

  • 配置文件为_config.yml

  • 修改最后的deploy信息

    deploy: type: 'git' repo: https://github.com/Liyulive/liyulive.git branch: master

  • 安装npm git

    npm install hexo-deployer-git --save

  • 部署

    hexo clean hexo g hexo d

    此时已经可以通过用户名.github.io来访问博客了。

使用 Netlify

  1. 注册Netlify账号
  2. Site - Import From Git
  3. 登录github账号，导入仓库

域名设置

  1. 获取域名（可以通过Freenom免费获取域名）
  2. 点击Domain settings
  3. 设置域名，添加解析

Enjoy!
