使用Hexo+Github+Netlify搭建博客

前置准备

  • 安装Node.js
  • 安装git

创建GitHub仓库

  1. 创建仓库 用户名.github.io

  2. 生成ssh密钥文件

  1. 打开Github Keys页面

​ 新建SSH密钥，将id_rsa.pub的全部内容复制进去。

局部安装Hexo

  1. 安装到特定目录
  1. 添加环境变量（非必要）

​ 将目录\node_modules\.bin添加至系统环境变量下。如不添加环境变量则使用npx hexo init等命令。

  1. 初始化Hexo

配置Hexo

  • 配置文件为_config.yml
  • 修改最后的deploy信息
  • 安装npm git
  • 设置git忽略ssl
  • 设置git代理（非必须）
  • 部署

​ 此时已经可以通过用户名.github.io来访问博客了。

使用 Netlify

  1. 注册Netlify账号
  2. Site - Import From Git
  3. 登录github账号，导入仓库

域名设置

  1. 获取域名（可以通过Freenom免费获取域名）
  2. 点击Domain settings
  3. 设置域名，添加解析

Enjoy!
